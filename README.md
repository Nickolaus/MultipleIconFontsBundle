# MultipleIconFontsBundle

SymfonyBundle using multiple icon fonts + Twig Function

Currently bundled icon-font-sets:

   - [FontAwesome]
   - [Glyphicons]
   - [MaterialIcons]
   - [Ionicons]
   

## Usage:
   - in progress


### Planned Features:
   - octicons intetration
   - foundation-icons interation


### Known Issues:
